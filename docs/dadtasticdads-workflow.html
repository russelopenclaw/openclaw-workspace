Key Improvements

1. Joke Format Detection + Audio-Synced Transitions 🎯

Current: 60/40 Time Split

Guesses when punchline lands. Accuracy: ~40%

Treats all jokes the same (setup/punchline only)

Proposed: Format Detection + Timing

 Step 1: Detect joke format
 ┌─────────────────────────────────────┐
 │ "Waking up was eye-opening"         │ → One-liner
 │ "Why...? Because..."                │ → Setup/Punchline  
 │ "Doctor: I need help... I don't..." │ → Multi-line
 └─────────────────────────────────────┘

 Step 2: Split with timing
 {
   format: "setup-punchline",
   segments: [
     { text: "Why can't your nose be 12\"?", 
       atPercent: 0.0, display: "fade-in" },
     { text: "Because then it'd be a foot!", 
       atPercent: 0.68, display: "pop-in" }
   ]
 }

Format Examples:

One-Liner: "Waking up this morning was an eye opening experience."
  • → Single text block
  • → Fade in at 0:01, hold entire duration
  • → No timing split needed
Setup/Punchline: "Why can't your nose be 12 inches long? Because then it'd be a foot!"
  • → Two segments (question → answer)
  • → Setup fades in, punchline replaces at detected timing
  • → Typical split: 60-70% through audio
Multi-Line/Conversational: "Doctor you've got to help me, I'm addicted to Twitter. Doctor: I don't follow you."
  • → 3+ segments or dialogue exchanges
  • → Progressive reveals or alternating text blocks
  • → Each segment timed to natural speech pauses
